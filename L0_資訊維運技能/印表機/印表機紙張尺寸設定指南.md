## 1. 常用紙張尺寸對照

### 國際標準（ISO A 系列）

| 名稱 | 尺寸（mm） | 用途 |
|------|-----------|------|
| A3 | 297 × 420 | 大圖、海報、工程圖 |
| **A4** | **210 × 297** | **辦公室最常用** |
| A5 | 148 × 210 | 便條紙、小冊子 |
| A6 | 105 × 148 | 明信片大小 |

### 美規（Letter 系列）

| 名稱 | 尺寸（mm） | 尺寸（英吋） | 用途 |
|------|-----------|------------|------|
| **Letter** | **215.9 × 279.4** | **8.5 × 11"** | 美國辦公室標準 |
| Legal | 215.9 × 355.6 | 8.5 × 14" | 法律文件 |
| Tabloid | 279.4 × 431.8 | 11 × 17" | 相當於 A3 |

### 台灣慣用切法（菊版系列）

| 俗稱 | 正式名稱 | 尺寸（mm） | 對應 ISO |
|------|---------|-----------|---------|
| 全開 | 菊全 | 636 × 939 | 約 A1 |
| 對開 | 菊對 | 469 × 636 | 約 A2 |
| **中一刀（4開）** | **菊四** | **318 × 469** | **約 A3** |
| 8開 | 菊八 | 234 × 318 | 約 A4 |
| 16開 | 菊十六 | 159 × 234 | 約 A5 |

## 2. 中一刀是什麼？

「**一刀**」在台灣印刷業是指**將全張紙對半裁切一次**的動作。「中一刀」即以菊版（636×939mm）為基礎，沿中線裁切一次，得到的尺寸。

```
菊全 636 × 939 mm
  │
  ├─ 中一刀（菊對開）→ 469 × 636 mm（約 A2）
  │     │
  │     ├─ 再中一刀（菊四開）→ 318 × 469 mm（約 A3）
  │     │     │
  │     │     └─ 再中一刀（菊八開）→ 234 × 318 mm（約 A4）
```

### 在辦公室語境中

> 「拿一疊**中一刀**的紙來」通常指 **A4 尺寸的影印紙**（因為影印紙本來就是按 A4 裁好的）。

但在**印刷、表單設計**語境中，「中一刀」可能指不同的尺寸，需依上下文判斷。

### 訂購紙張時的注意事項

- 購買影印紙：直接說「A4 70g 影印紙」，避免混淆
- 購買連續報表紙（點陣機）：需指定寬度（如 9.5 吋、14.5 吋）及行數（如 11 吋高）

## 3. A4 vs Letter 的差異與常見問題

### 尺寸差異

```
Letter (215.9 × 279.4)      A4 (210 × 297)
┌──────────────┐            ┌─────────────┐
│              │            │             │
│  Letter      │            │     A4      │
│  較寬 +5.9mm │            │  較長+17.6mm│
│              │            │             │
└──────────────┘            └─────────────┘
```

| 比較 | Letter | A4 |
|------|--------|-----|
| 寬度 | 215.9 mm（較寬） | 210 mm |
| 高度 | 279.4 mm（較短） | 297 mm（較長） |
| 常見地區 | 美國、加拿大 | 台灣、歐洲、全球大多數地區 |

### 設定錯誤的常見症狀

| 症狀 | 原因 | 解決方法 |
|------|------|---------|
| 列印後下方被截掉 | 驅動設 Letter，紙放 A4（高度不夠） | 驅動改為 A4 |
| 左右有多餘白邊 | 驅動設 A4，紙放 Letter（寬度不夠） | 驅動改為 Letter |
| PDF 顯示正常但印出來跑版 | PDF 是 Letter，印表機預設 A4 | Adobe Reader 列印時選「縮放頁面以符合紙張」 |
| Word 文件印出來有白框 | Word 文件設 Letter，印表機是 A4 | Word → 版面設定 → 紙張 → 改為 A4 |

### 強制縮放（緊急時使用）

當無法修改原始文件時，可在列印對話框：
- Adobe Reader：「縮放頁面以符合紙張（Fit to Printable Area）」
- Word：列印 → 縮放至紙張大小 → 選 A4 或 Letter
- Chrome 列印：更多設定 → 縮放 → 調整 %

## 4. 紙張設定的兩個層級

**常見誤解**：改了應用程式的紙張大小，為什麼印出來還是不對？

原因：紙張設定有**兩個獨立的地方**，兩者都要一致。

```
層級 1：印表機驅動（系統層）
    控制台 → 裝置和印表機 → 右鍵印表機
    → 印表機內容 → 進階 → 預設值 → 紙張尺寸

層級 2：應用程式（文件層）
    Word / Excel / PDF → 列印設定 → 紙張大小
```

### 設定步驟（確保兩層一致）

**Step 1：設定驅動預設紙張**

1. 控制台 → 裝置和印表機
2. 對印表機右鍵 → **列印喜好設定**
3. 找到「紙張尺寸」或「Paper Size」→ 選 **A4**
4. 套用 → 確定

**Step 2：設定應用程式紙張**

- **Word**：版面設定 → 紙張 → A4
- **Excel**：版面配置 → 大小 → A4
- **Adobe PDF**：檔案 → 列印 → 頁面設定 → A4
- **記事本 / 一般程式**：檔案 → 版面設定 → 紙張大小 → A4

### 實體紙匣設定（部分印表機）

多紙匣印表機（如事務機）需額外確認：

1. 印表機內建面板 → 紙匣設定 → 確認各紙匣的紙張尺寸登記正確
2. 若紙匣顯示「Letter」但實際放 A4，印表機會要求確認或報錯

## 5. 標籤機紙張設定技巧

標籤機（TSC、SATO 等）沒有「A4/Letter」概念，使用**自訂尺寸**。

### 標籤尺寸設定位置

```
控制台 → 裝置和印表機 → 右鍵標籤機 → 列印喜好設定
  └─ 紙張標籤（Paper）
       ├─ 標籤寬度（Label Width）：mm
       ├─ 標籤高度（Label Height）：mm
       └─ 感應器類型（Sensor Type）
```

### 感應器類型說明

| 感應器類型 | 說明 | 適用標籤 |
|-----------|------|---------|
| **Gap（間距）** | 偵測標籤間的透明間隙 | 一般離型紙標籤（最常見） |
| **Black Mark（黑標）** | 偵測底部黑色定位標記 | 有黑色線條的連續標籤紙 |
| **Continuous（連續）** | 不偵測，按設定高度切割 | 無間距的熱感紙捲 |

### 標籤跑位排查

| 症狀 | 原因 | 解決方法 |
|------|------|---------|
| 每張都往下偏移 | 標籤高度設定比實際小 | 加大標籤高度數值 |
| 越印越偏 | 感應器類型錯誤 | 改感應器類型，重新校正（Calibrate）|
| 第一張正常，後面跑掉 | 需要校正感應器 | 驅動或面板執行 Calibration |
| 印出來超出標籤邊緣 | 寬度設定超過標籤實際寬度 | 縮小寬度數值，重新測量標籤 |

### 校正感應器（Calibration）

**TSC**：
- 面板按 `Feed` 鍵直到指示燈閃爍（各型號略有差異）
- 或 TSC Console → Calibrate Sensor

**SATO**：
- 面板 → Maintenance → Sensor Calibration

> 更換標籤紙（不同廠牌、不同尺寸）後，**必須重新校正**。

## 6. 常見跑版問題排查

### 快速診斷流程

```
列印結果不對
  │
  ├─ 整體縮小或放大？
  │     └─ 應用程式「縮放至紙張」被勾選 → 取消縮放
  │
  ├─ 內容被截掉（底部或右側）？
  │     └─ 紙張尺寸設錯（A4/Letter 混淆）→ 統一尺寸
  │
  ├─ 有大片空白邊？
  │     └─ 邊距設定過大，或紙張比文件大 → 調整邊距 or 紙張
  │
  ├─ 標籤機每張偏移越來越大？
  │     └─ 感應器未校正 → 執行 Calibration
  │
  └─ 點陣機列印位置偏上或偏下？
        └─ 表單行距與紙張高度不符 → 調整程式輸出行數或換紙
```

### 邊距調整技巧

部分舊印表機可列印範圍（Printable Area）無法到達紙張邊緣，強制靠邊會被截掉：

- **Word**：版面設定 → 邊界 → 設定合理邊距（建議最小 10mm）
- **標籤軟體（BarTender）**：頁面設定 → 確認邊界不超過可列印區域
- 驅動屬性中可查詢「可列印區域」確認最小邊距限制
