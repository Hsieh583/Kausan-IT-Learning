## 1. 兩種不同的「權限」

印表機相關的「權限」分兩個層次，常被混淆：

| 層次 | 說明 | 由誰管理 |
|------|------|---------|
| **系統權限** | 能否「安裝驅動程式」到 Windows | Windows 帳號類型（管理員/標準用戶）|
| **印表機使用權限** | 能否「使用 / 管理」已安裝的印表機 | 印表機安全性設定 |

## 2. 安裝驅動需要的系統權限

### Windows 帳號類型

| 帳號類型 | 能安裝驅動？ | 說明 |
|---------|-----------|------|
| **系統管理員（Administrator）** | 可以 | 完整系統控制權 |
| **標準使用者（Standard User）** | 不行 | 需要 UAC 提示，輸入管理員密碼 |
| **來賓（Guest）** | 不行 | 最低權限，幾乎什麼都不能做 |

### UAC（使用者帳戶控制）提示

用標準帳號執行驅動安裝時，會出現：

```
┌──────────────────────────────────────┐
│  您要允許這個應用程式變更您的裝置嗎？  │
│                                      │
│  [應用程式名稱]                       │
│  已驗證發行者：[廠商名稱]              │
│                                      │
│  [是] [否]                           │
└──────────────────────────────────────┘
```

- 若登入帳號是**管理員**：直接點「是」即可
- 若登入帳號是**標準用戶**：需輸入管理員帳號的密碼才能繼續

### 確認自己的帳號類型

```cmd
:: 方法一：命令提示字元
net user %username%
:: 找「Local Group Memberships」這行，若有 *Administrators 即為管理員

:: 方法二：直接查看
控制台 → 使用者帳戶 → 查看帳戶類型（管理員 / 標準使用者）
```

### 以管理員身份執行安裝程式

標準用戶要安裝驅動時：

1. 對安裝程式（`.exe`）**右鍵**
2. 選「**以系統管理員身份執行**」
3. 輸入管理員帳號的密碼
4. 繼續安裝

> 如果不知道管理員密碼，請聯絡 IT 部門協助安裝。

### Point and Print（網路印表機自動安裝）

若印表機是透過 Windows 伺服器共享，一般用戶連線時可能自動安裝驅動，不需管理員介入。但需伺服器端先行設定 Point and Print 原則。

## 3. 印表機本身的使用權限

已安裝的印表機可以設定「誰能用、能做什麼」。

### 三種印表機權限

| 權限 | 說明 | 預設擁有者 |
|------|------|----------|
| **列印（Print）** | 送出列印工作 | Everyone（所有人）|
| **管理這台印表機（Manage Printer）** | 暫停/繼續印表機、更改設定 | Administrators |
| **管理文件（Manage Documents）** | 刪除/暫停別人的列印佇列 | Creator Owner + Administrators |

### 查看與修改印表機權限

1. 控制台 → 裝置和印表機
2. 對印表機右鍵 → **印表機內容**
3. 點選 **安全性（Security）** 標籤
4. 可看到使用者/群組清單及其對應權限

```
┌─────────────────────────────────────────┐
│ 群組或使用者名稱：                        │
│  Everyone                               │
│  CREATOR OWNER                          │
│  Administrators                         │
│                                         │
│ Everyone 的權限：          允許  拒絕    │
│  列印                       ☑    □     │
│  管理這台印表機               □    □     │
│  管理文件                    □    □     │
└─────────────────────────────────────────┘
```

### 常見需求設定

**情境 A：只允許特定部門使用某台印表機**
1. 安全性 → 移除 `Everyone` 的列印權限
2. 新增該部門的 Windows 群組（如 `倉庫人員`）
3. 給予「列印」權限

**情境 B：防止一般用戶刪除別人的列印工作**
1. 確認 `CREATOR OWNER` 有「管理文件」，`Everyone` 沒有
2. 如此每人只能管理自己送出的工作

**情境 C：讓特定人員可以暫停/重啟印表機**
1. 新增該用戶到安全性清單
2. 勾選「管理這台印表機」

## 4. 多用戶電腦的印表機管理

### 系統印表機 vs 用戶印表機

| 類型 | 安裝方式 | 可見對象 | 需要管理員？ |
|------|---------|---------|-----------|
| **系統印表機** | 透過系統管理員安裝 | **所有用戶** 登入後都看得到 | 是 |
| **用戶印表機** | 用戶自行安裝（有限制） | **只有該用戶** 登入後看得到 | 否（但有限制）|

> 倉庫電腦、共用工作站：建議由管理員安裝**系統印表機**，確保所有登入帳號都能使用。

### 多用戶場景常見問題

**問題**：A 同事安裝了印表機，B 同事登入同一台電腦卻看不到。

**原因**：A 安裝時是以用戶層級安裝，不是系統層級。

**解法**：
1. 以管理員帳號登入（或 UAC 提升）
2. 重新安裝印表機驅動（管理員身份執行）
3. 確認安裝完成後，所有用戶登入都可看到

---

**問題**：更換 Windows 帳號後，印表機預設值（紙張、黑白/彩色）跑掉。

**原因**：「列印喜好設定」預設值分**每個用戶獨立**儲存，不是全電腦共用。

**解法**：
- 每個用戶需各自設定一次「列印喜好設定」
- 或由管理員設定驅動的「全系統預設值」：
  1. 右鍵印表機 → **印表機內容**（不是「列印喜好設定」）
  2. 進階標籤 → **預設值（Defaults）** → 設定後，新用戶預設會套用此設定

---

**問題**：標準用戶無法更改印表機設定（紙張大小、黑白等）。

**原因**：「管理這台印表機」權限未開放。

**解法**（適用確需讓一般用戶自己調整設定的場景）：
1. 安全性 → 找到該用戶或 `Everyone`
2. 勾選「管理這台印表機」
3. 注意：此操作讓用戶可以暫停整台印表機，需評估風險

---

### 共用電腦的建議設定

| 設定項目 | 建議做法 |
|---------|---------|
| 驅動安裝 | 由 IT 管理員統一安裝為系統印表機 |
| 列印權限 | Everyone 保留「列印」，一般情況不需修改 |
| 管理權限 | 只開放給 IT 帳號或主管帳號 |
| 預設紙張 | 管理員設定驅動預設值，避免每人各自設定 |
| 預設印表機 | 各用戶可自行設定，不影響其他人 |

## 5. 常見權限問題與解法

| 錯誤訊息 / 症狀 | 原因 | 解法 |
|----------------|------|------|
| 「存取被拒絕」（Access Denied）安裝時 | 非管理員帳號 | 右鍵 → 以系統管理員身份執行 |
| 「操作無法完成，存取被拒絕」刪除印表機時 | 非管理員 | 同上，或請 IT 協助 |
| 安裝成功但其他帳號看不到印表機 | 用戶層級安裝 | 管理員重新安裝 |
| 無法刪除佇列中的列印工作 | 沒有「管理文件」權限 | 重啟 Print Spooler 或請管理員清除 |
| 每次登入印表機設定都不見 | 列印喜好設定未儲存或帳號問題 | 管理員設驅動預設值 |
| 「Windows 無法連線到印表機」共用時 | 對方電腦的印表機分享權限未開放 | 對方電腦 → 印表機 → 安全性 → 開放 Everyone 列印 |

### 重啟 Print Spooler（解決卡住的佇列）

```powershell
# 以系統管理員身份執行 PowerShell
net stop spooler
del /Q /F /S "%systemroot%\System32\spool\PRINTERS\*"
net start spooler
```
