## 2. 找不到印表機 IP 怎麼辦

> 工廠內多台同型號印表機（如多台 WF-T810），IP 難以辨識時，依以下方法逐步排查。

---

### 方法一：從印表機面板直接查詢（最快）

**EPSON WF-T810（工廠常用型號）**

1. 按印表機面板上的 **首頁（Home）** 鍵
2. 進入 **設定（Settings）** → **網路設定（Network Settings）**
3. 選擇 **網路狀態（Network Status）**
4. 畫面上即顯示目前 IP 位址

或直接列印：
1. **設定 → 網路設定 → 列印狀態表（Print Status Sheet）**
2. 印出的紙張包含 IP、MAC 位址、連線狀態

> **多台機器辨識技巧**：拿著列印出的狀態表，對照機器上黏貼的資產標籤（建議 IT 在每台機器貼上 IP + MAC 標籤）。

---

### 方法二：EpsonNet Config（EPSON 官方掃描工具，推薦多台管理用）

> 適合工廠有多台 EPSON 印表機，一次找出全部 IP。

**下載**：https://www.epson.com.tw → 搜尋「EpsonNet Config」

**步驟**：
1. 安裝並開啟 **EpsonNet Config**
2. 點選 **Tools → EpsonNet Config（掃描模式）**，或直接執行 EpsonNet Setup
3. 軟體自動掃描區域網路，列出所有 EPSON 裝置，顯示：
   - IP 位址
   - MAC 位址
   - 型號名稱
   - 連線狀態
4. 點選某台裝置可直接進入設定頁面（修改 IP、設定靜態 IP）

```
範例結果：
IP: 192.168.1.51  MAC: xx:xx:xx:xx:xx:01  型號: WF-T810  狀態: Ready
IP: 192.168.1.52  MAC: xx:xx:xx:xx:xx:02  型號: WF-T810  狀態: Ready
IP: 192.168.1.53  MAC: xx:xx:xx:xx:xx:03  型號: WF-T810  狀態: Offline
```

---

### 方法三：Advanced IP Scanner（通用網路掃描，所有品牌適用）

> 免費工具，可掃描整段網路找出所有連線裝置。

**下載**：https://www.advanced-ip-scanner.com/tw/

**步驟**：
1. 安裝後輸入網段範圍，例如 `192.168.1.1-254`
2. 點選 **掃描（Scan）**
3. 在結果列表中篩選 **製造商（Manufacturer）** 欄位，找 `EPSON` 或 `Seiko`
4. 對應的 IP 即為印表機 IP

**搭配使用技巧**：
- 右鍵點選找到的 EPSON 裝置 → **開啟網頁（Open in browser）**
- 可進入印表機內建 Web 管理介面，確認型號與序號，對應到實體機台

---

### 方法四：cmd 快速掃描（不需安裝工具）

```cmd
:: 對整段網路送出 ping，讓 ARP 快取記錄回應的 IP
for /L %i in (1,1,254) do ping -n 1 -w 50 192.168.1.%i > nul

:: 查看 ARP 快取，找出 EPSON MAC 前綴（00:26:AB 或 44:D2:44 等）
arp -a
```

> EPSON 常見 MAC 前綴：`00:26:AB`、`44:D2:44`、`AC:3A:7A`
> 在 ARP 清單中比對前綴，即可縮小目標 IP 範圍。

---

### 根本解法：設定靜態 IP（強烈建議）

工廠多台同型號印表機，每次重開機 DHCP 可能配到不同 IP，導致安裝後隔天又連不到。

**解決方式：用 MAC 綁定 IP（DHCP Reservation）**

1. 取得印表機 MAC 位址（從狀態頁或 EpsonNet Config）
2. 登入路由器 / DHCP 伺服器管理介面
3. 新增 DHCP 保留：`MAC → 固定 IP`
4. 重新啟動印表機，確認取得固定 IP

**或在印表機本機設定靜態 IP（WF-T810）**：
1. 面板 → **設定 → 網路設定 → 進階設定 → TCP/IP**
2. 將 IP 設定方式改為 **手動（Manual）**
3. 輸入 IP、子網路遮罩、預設閘道

> **建議命名規則**（貼在機台上）：
> ```
> WF-T810 #3
> IP: 192.168.1.53
> MAC: xx:xx:xx:xx:xx:03
> 位置: 工廠 B 區包裝線
> ```

## 3. 租賃事務機（一鍵安裝）

> 廠商已為各倉庫準備打包安裝檔，IT 毋需手動安裝驅動。

### 安裝步驟

1. 向廠商索取對應倉庫的一鍵安裝包（`.exe` 或 `.pkg`）
2. 以**系統管理員身份**執行安裝包
3. 依畫面指示完成，安裝完成後印表機會自動出現於裝置清單
4. 列印測試頁確認正常

### 注意事項

- 每個倉庫的安裝包**不通用**，請確認版本對應正確倉庫
- 若廠商安裝包失效，聯絡廠商服務窗口重新取得
- 安裝包存放位置：`\\IT-Server\Printers\租賃事務機\<倉庫名稱>\` *(請依實際路徑修改)*

## 4. HP 無限印表機

> 適用 HP Ink Tank / Smart Tank 系列（無限供墨印表機）

### 前置準備

- 下載 **HP Smart App**：https://support.hp.com/tw-zh/drivers
- 搜尋型號後下載對應 Windows 驅動

### USB 安裝

1. 先**不要**插 USB 線，執行驅動安裝程式
2. 安裝程式提示時再插入 USB 線
3. 等待驅動自動辨識完成
4. 列印測試頁

### 網路安裝（IP 直連）

1. 在印表機面板確認 IP 位址（通常在「設定 → 網路資訊」）
2. 開啟 **控制台 → 裝置和印表機 → 新增印表機**
3. 選擇「印表機不在清單中」→「使用 TCP/IP 位址或主機名稱新增印表機」
4. 輸入印表機 IP，勾選「自動偵測印表機驅動程式」
5. 若自動偵測失敗，手動選擇對應型號驅動

### 常用型號驅動位置

| 型號 | 驅動備份路徑 |
|------|------------|
| HP Smart Tank 515 | `\\IT-Server\Printers\HP\Smart_Tank_515\` |
| *(其他型號請補充)* | |

## 5. EPSON L680 印表機

> 適用 EPSON L 系列無限供墨多功能印表機

### 前置準備

- 下載驅動：https://www.epson.com.tw → 支援 → 下載驅動程式
- 搜尋「L680」，選擇 Windows 版本下載

### USB 安裝

1. 執行 EPSON 驅動安裝程式，**安裝前不要插 USB**
2. 出現提示後插入 USB 線
3. 等待安裝完成，重新啟動電腦
4. 列印測試頁確認

### 網路安裝（IP 直連）

1. 在印表機面板列印「網路狀態頁」取得 IP
2. 進入**控制台 → 裝置和印表機 → 新增印表機**
3. 選「使用 TCP/IP 位址新增」，輸入印表機 IP
4. 選擇 EPSON L680 驅動（若無則從下載的驅動包選取）
5. 命名印表機（建議格式：`EPSON-L680-<地點>`）

### 掃描功能安裝

- 需另外安裝 **Epson Scan 2** 才能使用掃描功能
- 同樣從 EPSON 官網下載

## 6. EPSON LQ3500 點陣印表機

> 適用倉庫出貨單、多聯式表單列印

### 前置準備

- 下載驅動：EPSON 官網搜尋「LQ-3500」
- 選擇對應 Windows 版本（注意：Windows 10/11 需確認相容性）

### USB 安裝

1. 執行驅動安裝程式（以管理員身份）
2. 提示時插入 USB 線
3. 完成後確認裝置管理員無錯誤

### 並列埠安裝（舊式 LPT 連線）

1. **控制台 → 裝置和印表機 → 新增印表機**
2. 選「新增本機印表機」→ 連接埠選 `LPT1`
3. 手動選取 EPSON LQ-3500 驅動
4. 若找不到驅動，使用相容型號（如 LQ-3500+）

### 紙張設定重點

- 連續報表紙：進紙方式選「拖紙器」
- 多聯式表單：調整列印濃度（驅動設定 → 品質 → 草稿/高速）

## 7. TSC 工業標籤機

> 適用條碼標籤、出貨標籤列印，常見型號：TSC TTP-244、TSC TTP-343

### 前置準備

- 下載 **TSC Console**（管理工具）+ 驅動：https://www.tscprinters.com → 支援下載
- 或使用 **BarTender** / **NiceLabel** 等標籤軟體（視現有授權）

### USB 安裝

1. 執行 TSC 驅動安裝包
2. 插入 USB 線，系統自動辨識
3. 開啟 TSC Console 確認連線狀態（顯示機器資訊表示成功）

### 網路安裝（TCP/IP）

1. 在印表機面板或使用 TSC Console 確認印表機 IP
   - TSC Console → 搜尋裝置 → 取得 IP
2. **控制台 → 裝置和印表機 → 新增印表機**
3. 選「使用 TCP/IP 位址新增」→ 輸入 IP，連接埠 `9100`
4. 選擇對應 TSC 型號驅動
5. 命名格式建議：`TSC-TTP244-<地點>`

### 標籤尺寸設定

1. 印表機屬性 → 列印喜好設定
2. 設定標籤寬度、高度（單位：mm）
3. 設定感應器類型（Gap 間距感應 / Black Mark 黑標感應）

## 8. SATO 熱感標籤機

> 適用熱感標籤（不需色帶），常見型號：SATO CG4 系列、SATO WS4 系列

### 前置準備

- 下載驅動：https://www.sato-global.com → Support → Downloads
- 搜尋對應型號，下載 Windows 驅動

### USB 安裝

1. 執行 SATO 驅動安裝程式
2. 選擇連線類型：USB
3. 插入 USB 線，完成安裝
4. 在驅動設定中確認標籤尺寸與感應器類型

### 網路安裝（TCP/IP）

1. 從印表機面板列印設定頁取得 IP
2. **控制台 → 裝置和印表機 → 新增印表機**
3. 選「使用 TCP/IP 位址新增」→ 輸入 IP，連接埠 `9100`
4. 選擇 SATO 對應型號驅動
5. 驅動安裝後在「列印喜好設定」中設定：
   - 標籤尺寸（寬 × 高，單位 mm）
   - 感應器：Gap（間距）或 Black Mark（黑標）
   - 列印速度與濃度

### 熱感 vs 熱轉印

| 模式 | 色帶 | 適用標籤 |
|------|------|---------|
| 熱感（Direct Thermal） | 不需要 | 短效期標籤（出貨單、臨時標） |
| 熱轉印（Thermal Transfer） | 需要 | 長效期標籤（資產標、永久條碼） |

## 9. OI 印表機伺服器

> OI Print Server 讓一台 USB 印表機透過網路共享給多台電腦使用

### 設定印表機伺服器

1. 確認 OI Print Server 已連上網路，取得 IP（參考管理介面或 DHCP 清單）
2. 登入 OI 管理介面（預設：`http://<Print Server IP>`）
3. 確認 USB 印表機已辨識（Port 1 顯示印表機型號）
4. 記下 Print Server IP（如 `192.168.1.xxx`）

### 在電腦上安裝（TCP/IP Port）

1. 先在電腦安裝對應印表機的驅動（不連接 USB）
2. **控制台 → 裝置和印表機 → 新增印表機**
3. 選「新增本機印表機」→「建立新的連接埠」→ 選 **Standard TCP/IP Port**
4. 輸入 Print Server IP，連接埠名稱自動填入
5. 選擇對應的印表機驅動
6. 完成後列印測試頁

### 多台電腦共用同一 OI 伺服器

- 每台電腦重複上述步驟，輸入相同的 Print Server IP 即可
- 建議命名格式：`<印表機型號>-OI-<地點>`（例：`LQ3500-OI-倉庫A`）

## 10. 人員調動 — 重新安裝本地印表機

> 人員異動至不同地點時，因 IP 不同，需重新設定印表機連線。

### 步驟一：確認新地點的印表機 IP

- 從印表機面板列印「網路設定頁」，或
- 詢問目的地 IT 窗口取得 IP 清單（參考：`\\IT-Server\Printers\各地IP清單.xlsx`）

### 步驟二：刪除舊印表機

1. 控制台 → 裝置和印表機
2. 對舊印表機右鍵 → **移除裝置**
3. 若無法移除：開啟「列印佇列管理員」服務，清除佇列後再刪除

### 步驟三：安裝新地點印表機

- **方法 A（推薦）**：重新執行對應型號的安裝步驟（參考本文件各章節），輸入新 IP
- **方法 B（快速）**：對現有印表機右鍵 → 印表機內容 → 連接埠 → 設定連接埠 → 更改 IP 為新地點 IP

### 步驟四：設為預設印表機

- 對新安裝的印表機右鍵 → **設定為預設印表機**

### 各地點印表機 IP 對照表

> *(由 IT 維護，請依實際環境填入)*

| 地點 | 印表機型號 | IP 位址 | 備註 |
|------|-----------|---------|------|
| 倉庫 A | EPSON L680 | 192.168.1.xxx | |
| 倉庫 A | TSC TTP-244 | 192.168.1.xxx | |
| 倉庫 B | SATO CG4 | 192.168.2.xxx | |
| 辦公室 | HP Smart Tank | 192.168.0.xxx | |
| *(依實際補充)* | | | |

## 11. 常見問題排查

| 症狀 | 可能原因 | 解決方法 |
|------|---------|---------|
| 印表機顯示離線 | IP 變更或網路斷線 | 確認 IP，重新設定 TCP/IP 連接埠 |
| 列印卡在佇列 | 佇列卡住 | 重啟 Print Spooler 服務（`services.msc`） |
| 驅動安裝失敗 | 權限不足 | 以系統管理員身份重新執行安裝程式 |
| 標籤尺寸跑掉 | 驅動紙張設定不符 | 印表機內容 → 列印喜好設定 → 重新設定標籤尺寸 |
| 點陣機列印亂碼 | 字元集設定錯誤 | 確認驅動語言設定（繁中環境選 CP950） |
| OI 伺服器找不到 | DHCP 更換 IP | 重新登入 OI 管理介面確認新 IP，更新連接埠設定 |
| 熱感標籤全黑 | 感應器類型錯誤 | 驅動設定 → 感應器改為 Gap 或 Black Mark |

### 重啟 Print Spooler 快速指令

```powershell
# 以系統管理員身份執行 PowerShell
net stop spooler
del /Q /F /S "%systemroot%\System32\spool\PRINTERS\*"
net start spooler
```
