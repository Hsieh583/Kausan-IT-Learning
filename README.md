# Kausan IT Learning

> IT 人員技術培訓文件庫 — 以 L1–L5 營運成熟度框架為核心，支援 ISO 27001 認證準備。

---

## 專案背景

本專案為 Kausan IT 部門的技術培訓知識庫，採用**營運優先（Operational-First）**策略：

> 以不中斷、不爆炸為核心，逐層提升 IT 品質

**2026 Q1–Q2 關鍵背景**

| 指標 | 現況 |
|------|------|
| 人力負載 | 205%（1 人承擔 5 人工作） |
| 核心系統故障風險 | 90%（3 個月內） |
| ISO 27001 認證窗口 | 2–6 月（剩 4 個月） |
| 培訓總時數目標 | 300 小時（Q1–Q2） |

---

## L1–L5 成熟度框架

```
L5: 文件與合規        ← 最終驗收層（ISO 27001）
     ↑
[證據自動化流水線]    ← 橋接層：自動生成稽核登記簿
     ↑
L4: 工具與自動化      ← 加速交付、減少人工浪費
L3: 使用者體驗        ← 降低摩擦、提升服務順暢度
L2: 防禦與監控        ← 出事有訊號、能快速應對
L1: 可靠性與容災      ← 基礎：不中斷、不爆炸（DR/HA）
```

各層級成熟度目標（2026 Q1–Q2）：

| 層級 | 主題 | 起點 → 目標 |
|------|------|------------|
| L1 | 可靠性 / DR / 備份 | 1.5 → 2.5 |
| L2 | SIEM / 事件回應 / 弱點管理 | 1.5 → 2.5 |
| L3 | IAM 帳號管理 | 1.5 → 2.0 |
| L4 | 自動化工具（規劃中） | 1.5 → 2.0 |
| L5 | ISO 稽核 / Git 文件管理 | 2.0 → 3.0 |

## 文件慣例

- **語言**：繁體中文為主，英文為輔（雙語對照）
- **版本格式**：每份文件包含版本號、最後更新日期、負責人
- **成熟度評分**：1–5 分制，1 = 初始，5 = 卓越
- **優先級標示**：🔴 P0 緊急 / 🟠 P1 關鍵 / 🟡 P2 改善
